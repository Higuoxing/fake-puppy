# fake-puppy

> Yet another auth server for wifi-dog

### Dependency

* MongoDB
* Node.js

### Features

* Swap QR-Code to login
* Real-time monitor

### Deploy

```bash
git clone https://github.com/higuoxing/fake-puppy.git
cd fake-puppy
npm install
npm start
```
